Weekly newsletter mock

This preview shows the HTML email that would go to your team each Friday. Content is built from the same JSON and Key observations pipeline as the live dashboard.

  • Source data: static_home/data/etp_kpis.json, crypto_kpis.json, rwa_explore_asset_type.json
  • Generator: python scripts/build_weekly_newsletter.py
  • Email file: weekly-newsletter-email.html (inline CSS, ~640px width)

To refresh after a data export, run the generator from the repo root. Set SITE_BASE_URL if your GitHub Pages URL differs.
